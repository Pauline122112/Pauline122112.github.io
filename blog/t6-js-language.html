    Parentheses ()
    1.Used to control the order of an execution and to contain a list of parameters. eg. Execute a function var result = doSomething()
    Brackets []
    2. For an array of values: var names = [ 'Pauline', 'Jason', 'Christine'];
    Braces {}
    3. Braces or curly braces are the "{" and the "}" signs, they are required when building your own functions and "if" statements. Every opening brace, must have a closing brace so it is executed together.
    For example: if (age> 31) { console.log("You are old"); }
    Single Quotes ''
    4. Is the same as "Double Quotes", it is used to contain the same string:
    For example: var name = 'Pauline';
    Double quotes ""
    5. Is the same as 'Single Quotes', and is used to contain a string which is text inside quotes.
    For example: var name = "Pauline";
