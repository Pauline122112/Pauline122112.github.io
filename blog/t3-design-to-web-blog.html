    What's a responsive site? Why is it important?

    It's an approach to making website pages look great an any size. Devices, platforms and browser are expanding, therefore responsive sites are a way to stay current.

    What is mobile first design? Why is it important?

    Designing for smaller screens first, then add more features and content for bigger and bigger screens. If a site looks good on a mobile device, it can translate better to all devices.

    What are frameworks? Pros and Cons

    Pros

    • Speeds up mock-up process
    • Clean and tidy
    • Solution to CSS problems
    • Browser Compatibility

    Cons

    • Mixes content and presentation
    • Unused code leftover
    • Slower learning curve
    • Can't learn to do it yourself

    What is a Wireframe? Why is it important?

    When you plan the layout and interaction of an interface without being distracted by colours or typefaces. A wireframe is simple to create and should provide enough detail so that the client can understand how the website will be presented.

    What aspects of my wireframe I found hard to implement? Why?

    Alignment of the content. I'm still trying to figure out the sections, and still trying to piece together what goes where, but I'm confident that I will be in control and able to find a solution sufficiently.
